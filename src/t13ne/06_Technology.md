# Technology and Crafting

## Tech Progression
*   **Tech Levels:** Gated behind stories. Found items reveal positions on a "fog of war" tech tree.
*   **Compatibility:** Species technologies are not entirely cross-compatible (except Vemlek).
*   **Experimentation:** Players can repair/solder components. Potential for mini-games or discovering new tech.

## Component Definition
Components are defined by a bitwise ID number and efficiency ratings (A:Z).

### Material Bits
| Bit | Material | Meaning |
| :--- | :--- | :--- |
| 1 | Fuel | Consumed |
| 2 | Metal | Metallic element |
| 3 | Non-metal | Non-metallic element |
| 4 | Semi-conductor | Logic gates |
| 5 | Electrical Conductor | Carries electricity |
| 6 | Plasma waveguide | Carries Plasma |
| 7 | Polymer | Plastics |
| 8 | Organic | Biological |
| 9 | Radioactive | Radioactive |
| 10 | Psychoactive | Psychoactive |
| 11 | Catalyst | Required but not consumed |
| 12 | Nano-alloys | High tech |
| 13 | Crystalline | Electronics/High tech |
| 14 | Hyperdense | Extreme chemistry |
| 15 | Dark Matter | Non-baryonic |

Interactions and systems are actually defined as T13 Facet interactions and Annexes. Technologies are created by mixing two proficiencies into and ANnex in standard T13 fashion. This can create emergent technologies, but does also tie tech levels to T13 Sway table (which is probably not a bad thing). Technology is probably Bold in the universe of Wormhole Racers so tech can achieve magical things, and we have a lot of tech avaiable on the tech tree for T13.  Task Bits and so on may need to be compared against T13 Sway levels and so on during tech creation.
