# T13 Author's Guide

The **Author** creates the building blocks of the T13 universe: Plots, Worlds, Characters, and Knots.

## Author Workbench
Use the **Author Mode** in the Companion or the **Workbench** in the Engine Editor.

### Creating Components
- **Proficiencies**: The basic units of skill and power. Defined by a Facet, a Genre, and an Era.
- **Annexes**: Combinations of proficiencies. They can be Skills, Talents, or Powers.
- **Plots**: Create complex narrative hierarchies (Cycle -> Epic -> ... -> Scene). Use the **Cascade** feature to procedurally generate sub-plots.

### Narrative Loom
- Define the **Conflict** (Dominant vs. Pressed sides).
- Hook characters into specific facets of the conflict.
- Use the **AIService** (Ollama/Bitnet) to flesh out descriptions and titles based on T13 Gematria.
